# Basic Containerized Ruby on Rails App

Based on `rails new <app>`

# Versions
Ruby 3.4.2
Rails 8.0.1

# To run from Docker
## Run the following:
```
docker-compose up -d
```
## Navigate to: http://127.0.0.1:3000/hello

# To run locally without Docker
## Run the following:
```
bin/rails server
```
## Navigate to: http://127.0.0.1:3000/hello or http://localhost:3000
